# 2_Data_ACR_HMD

This folder includes the recordings of the head movements of each participant for the entire duration of the HMD exposure.
	
## File structure 

The columns contain a sequence of parameters in the following order:
  
	- TimeStamp (date)
	- TimeStamp (time)
	- MediaTime
	- PositionX
	- PositionY
	- PositionZ
	- RotationX
	- RotationY
	- RotationZ
	- RotationW
	- EulerX
	- EulerY
	- EulerZ
